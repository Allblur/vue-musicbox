# vue-musicbox

A musicbox app built with Vue 2.0 + vue-router + vuex, with server-side rendering.

## Build Setup

**Requires Node.js 6+**

``` bash
# install dependencies
npm install

# serve in dev mode, with hot reload at localhost:3333
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
